This repository uses shared AI instructions under `.ai/`.

Before making changes, read:

1. `.ai/project.md`
2. `.ai/rules.md`
3. `.ai/workflows.md`

## General rules

- Keep changes small and maintainable.
- Do not rewrite large modules unless explicitly requested.
- Explain assumptions before making risky changes.
- Check public API impact before editing exported types.
- Add or update tests when behavior changes.
- Prefer practical fixes over clever abstractions.
